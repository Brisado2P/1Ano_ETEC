Tags que já aprendemos

Elemento Descrição
Head Representa uma coleção de metadados sobre o documento, incluindo links suas definições de scripts e folhas de estilo.
ul Define uma lista não ordenada
ol Define uma lista ordenada de itens, que é uma lista que mudam o seu significado, se alterar a ordem dos seus elementos.
Body Representa o principal conteúdo de um documento HTML. Há apenas um elemento *body* em um documento.
Html Representa a raiz de um documento HTML ou XHTML. Todos os outros elementos devem ser descendentes desse elemento.
H1, H2, H3, H4, H5 e H6 São elementos que representam os seis níveis de titulos de cabeçalhos dos documentos. Um elemento título descreve brevemente o tema da seção.
img Representa uma imagem.
a Representa um hyperlink, ligando a outro recurso.
Title Define o título do documento , apresentado na barra de título do navegador ou na guia da página. Ele só pode conter texto e todas as tags contidas não são interpretados.
Meta Define os metadados que não podem ser definidas usando outro elemento HTML.
p Define uma parte que deve ser exibido como um parágrafo.
br Representa uma quebra de linha
tr Representa uma linha de células em uma tabela.
table Representa uma tabulação de dados - ex: uma tabela de informações expressada em duas dimensões através de linhas e colunas.
td Representa uma célula de dados em uma tabela.
th Representa uma célula de cabeçalho em uma tabela.

10 tags que não aprendemos

Elemento Descrição
footer Define o rodapé das seções ou da página.
header Define um grupo de títulos ou o cabeçalho de uma determinada seção.
nav Define um grupo ou bloco de links de navegação.
address Define uma seção que contém informações de contato, como endereço e telefone.
main Define o conteúdo principal ou importante no documento. Existe apenas um elemento -main- no documento.
time Representa um valor de data e hora, eventualmente com um equivalente legível por máquina.
var Representa uma variável, que pode ser uma expressão matemática, ou código de programação, um identificador representando uma constante, um símbolo identificando uma quantificação física um parâmetro de função ou um mero placeholder.
mark Representa texto destacado para fins de referência, que é por sua relevância em outro contexto.
object Representa uma fonte externa , a qual será tratada como uma imagem, um sub-documento HTML ou um recurso externo a ser processado por um plugin.
menu Representa uma lista de comandos.
